# captureone-icc-generator

Script used to create custom ICC input profiles for use in Capture One.

One way to use film emulations in Capture One is to select a custom ICC profile in the 'Base Characteristics' tool. This script takes a camera specific input profile and combines it with a film emulation to give a new camera-specific profile with the emulation embedded.

To use the script script you will need:

1. A working python3 install, with py-pillow and py-numpy packages
2. A camera input profile from Capture One. These can be found in the application install files or generated by using 'Save as ICC Profile...' from Capture One's Color Editor.
3. A film emulation in Hald CLUT format. There are numerous examples of these online, including a large collection developed for use with RawTherapee.

Pass the input profile to the python script, along with a strength parameter and an output name. For example:

  python3 FujiGFX100S-Generic.icm Kodachrome.png -s 75 -o "FujiGFX100S-Kodachrome 75%.icc"

Copy the resulting file to somewhere that Capture One can see it (eg: on macOS, "~/Library/ColorSync/Profiles"), restart Capture One, and then select the profile under the 'Base Charactistics' tab.

Please note that this script is largely AI generated, and was originally intended for a single conversion for a profile that I wanted to use across multiple RAW editors. Accordingly, use it with caution and test the result carefully.
